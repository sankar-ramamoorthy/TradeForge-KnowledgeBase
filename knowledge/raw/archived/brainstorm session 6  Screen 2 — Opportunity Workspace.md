# Screen 2 — Opportunity Workspace

This screen is where:

```text id="opp1"
raw market observations
become
structured trade hypotheses
```

This is one of the most important philosophical screens in TradeForge.

Why?

Because most trading systems completely skip this layer.

They jump directly from:

```text id="opp2"
“market moved”
→
“place trade”
```

But for a discretionary swing trader, the actual edge often exists in:

* interpretation
* patience
* filtering
* contextual thinking
* waiting for alignment
* knowing why NOT to trade

So this screen is fundamentally about:

```text id="opp3"
structured opportunity development
```

NOT:

```text id="opp4"
signal generation
```

---

# Core Purpose Of Screen 2

This screen answers:

```text id="opp5"
“What opportunities are forming,
and are they becoming valid trade candidates?”
```

This is pre-plan.
Pre-execution.
Pre-commitment.

It is the “thinking workspace.”

---

# Emotional Purpose

The screen should psychologically encourage:

* patience
* curiosity
* structured reasoning
* observational discipline
* context awareness

NOT:

* urgency
* FOMO
* compulsive execution

---

# MVP Swing Trader Reality

A swing trader constantly watches:

* leaders
* pullbacks
* breakouts
* macro themes
* sector rotation
* earnings setups
* failed moves
* volatility changes

Most of these never become trades.

That is GOOD.

This screen should normalize:

```text id="opp6"
unexecuted thinking
```

That is a huge conceptual shift.

# Regime Alignment

VERY important for swing trading.

Example:

```text id="opp25"
Regime Alignment

✓ Strong sector leadership
✓ Positive breadth
✓ Momentum regime favorable
⚠ Volatility elevated
```

This helps prevent:

```text id="opp26"
good setups in bad environments
```

---

# Risks / Invalidation

One of the most important sections.

Example:

```text id="opp27"
Invalidation Conditions

- Semis lose leadership
- Breadth deteriorates
- Failed breakout under 900
- CPI event disrupts market structure
```

---

# Conditions Required

This is critical psychologically.

Because it encourages waiting.

Example:

```text id="opp28"
Required Before Promotion To Plan

✓ Relative strength maintained
✗ Volume confirmation pending
✗ Market breadth confirmation pending
```

This is VERY different from:

```text id="opp29"
“BUY NOW”
```
