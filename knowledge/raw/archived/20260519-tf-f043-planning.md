# TF-F043 Planning - FMP Stable Fundamentals Endpoint Fix

## Diagnosed Root Cause

The configured FMP credential can access `/stable/income-statement`, but the
runtime adapter uses older `/api/v3/...` endpoints. The resulting provider
failure is surfaced correctly by the provider-attempt transparency path, but the
adapter route mapping is stale.

## Chosen Approach

Update `FmpFundamentalsProvider` to use:

- `https://financialmodelingprep.com/stable/profile?symbol={symbol}&apikey=...`
- `https://financialmodelingprep.com/stable/income-statement?symbol={symbol}&apikey=...`
- `https://financialmodelingprep.com/stable/ratios?symbol={symbol}&apikey=...`

Normalize the stable response into the existing `FundamentalsBundle` contract:

- profile: `companyName`, `sector`, `industry`
- income statement: `date`, `revenue`, `netIncome`
- ratios: `priceToEarningsRatio`; `return_on_equity` remains optional when FMP stable ratios do not expose an equivalent field in the observed payload

## Scope Boundary

The fix remains inside the FMP infrastructure adapter and tests. It does not
expand domain models or workspace displays to include FMP stable quote fields.
The quote endpoint observation remains a future candidate captured in the raw
brainstorm note.

## Event, Replay, And Lifecycle Implications

- Event impact: none.
- Replay impact: none; provider data remains advisory context and non-canonical.
- Lifecycle impact: none; fundamentals availability still cannot authorize or block lifecycle transitions.

## Testing Approach

- Update FMP adapter normalization tests for stable response field names.
- Add a URL-construction test proving the adapter calls `/stable/...` routes with `symbol` query parameters.
- Run focused fundamentals tests.
- Run broader relevant provider/context tests and frontend build if code touches API shape. Frontend shape is not expected to change.

## Open Questions

- Whether quote fields such as `marketCap`, `priceAvg50`, and `priceAvg200` should become a future price-context or instrument-summary enhancement.
