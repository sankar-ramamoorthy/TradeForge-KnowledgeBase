# Conflicting Evidence

Conflicting evidence is advisory context showing that linked observations do not
all point toward the same interpretation.

Conflict visibility helps operators notice ambiguity. It must not be collapsed
into an automatic decision, score, or recommendation.

## Runtime Stabilization

M13 implemented conflict summary and contradiction surfacing across opposing
thesis influence labels and conflict markers.

These summaries expose ambiguity for operator review; they do not resolve the
ambiguity into an authoritative action.
